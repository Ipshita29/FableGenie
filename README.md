# **FableGenie: AI Powered eBook Creator**

**FableGenie** is a full-stack, end-to-end platform built on the **MERN stack** that empowers users to create professional eBooks easily.  
Leveraging the power of **Google's Generative AI**, users can generate rich, structured content, which is then compiled into beautiful **PDF** and **Markdown** formats.

- **AI-Driven Content Generation**

- **Secure Authentication**  
  Implements **JWT (JSON Web Tokens)** for secure user registration, login, and authorization across all protected routes.

- **Rich eBook Generation**  
  Uses **PDFKit** to dynamically generate and export the created content into a professional, downloadable PDF document.

- **Markdown Editor**  
  Provides an integrated editor where generated or custom content is managed using **Markdown** for easy structuring.

- **File Uploads**  
  Uses **Multer** for handling file uploads (e.g., cover images, assets) to enrich the eBook creation process.
